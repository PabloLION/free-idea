# free-idea

Ideas formed in daily life — tracked as GitHub issues.

## Browse

All ideas are [GitHub issues](https://github.com/PabloLION/free-idea/issues).
Open issues are ideas worth exploring; closed issues are ideas that were
resolved, merged into other projects, or dropped.

## Contribute

Have an idea to add or feedback on an existing one? Open an issue or comment on
one.
