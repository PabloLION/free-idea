# Free Idea Project

## Local Issue Threads

Local issue threads are markdown files for tracking ideas and research that aren't published on GitHub.

### Naming Pattern

```
issue-{number}.{kebab-case-title}.md
```

**Examples:**
- `issue-53.zellij-layout-sharing-platform.md`
- `issue-51.vim-learning-game.md`

### Location

Store in `.dev/issues/` directory.
